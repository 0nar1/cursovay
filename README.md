# Информационная система академии «TOP» (Laravel + MySQL)

Полноценная версия сайта на Laravel с базой MySQL. Дизайн сохранен из демо-версии, функционал реализован для трех ролей:
- **Администратор** — управление курсами, расписанием, преподавателями и группами.
- **Педагог** — посещаемость, оценки, домашние задания, комментарии.
- **Студент** — расписание, домашние задания, посещаемость, оценки и средний балл, отзывы о курсах.

Демо-версия (HTML/CSS/JS) остается в корне проекта как референс для дизайна.

## Структура
- `laravel/` — основной проект Laravel (сервер + база).
- `BD.json` — исходные демо-данные для сидов.

## Запуск (Laravel)
1. Перейдите в `laravel/`
2. Установите зависимости:
   - `composer install --prefer-source --no-interaction --no-progress --no-dev`
3. Создайте ключ:
   - `php artisan key:generate`
4. Запустите миграции и сиды:
   - `php artisan migrate --seed`
5. Запуск сервера:
   - `php artisan serve`

## Демо-доступ
- **Админ:** `admin@top.local / admin`
- **Преподаватель:** `teacher1@top.local / teacher1`
- **Студент:** `student1@top.local / student1`

## База данных
Параметры подключения находятся в `laravel/.env` (MySQL).
